# Application réseau de management personnalisé - <br> héritage intellectuel technique (ARMP-HIT)

Application mobile développée en **Java** avec Android Studio permettant d’analyser et de visualiser les informations d’un réseau local.

<br> <br>

## Fonctionnalités

- **Analyse du réseau** — Récupération des informations essentielles du réseau.
- **Détection des appareils connectés** — Scan complet du sous-réseau pour identifier les appareils actifs.
- **Affichage du sous-réseau** — Présentation des résultats de détection préalablement enregistrés.

<br>
